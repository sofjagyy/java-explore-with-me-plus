# Feature: Comments

Реализован функционал комментариев к событиям.

1.  **Публичный доступ**:
    *   Просмотр комментариев к событию

2.  **Приватный доступ (для пользователей)**:
    *   Добавление комментария (только к опубликованным событиям).
    *   Редактирование своего комментария (`PATCH`).
    *   Удаление комментария (доступно автору комментария и инициатору события).
    *   Просмотр своих комментариев.

3.  **Административный доступ**:
    *   Удаление любого комментария (модерация).
